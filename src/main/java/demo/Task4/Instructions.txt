
## IterativeRefinement.java

Example: Add a method to refine the report

Vague Prompt:
    Add a method to refine the report

Iterative Refinement Prompt:
    Part 1:
    Add a method refineReport(String draft, String feedback, List<Integer> scores) that:
      - Applies one small, deterministic change per feedback round
      - Examples:
          - 'Add pass rate' → append pass rate with threshold 70
          - 'Make concise' → return a single-line summary
          - 'Return JSON' → convert report to JSON
     Part 2:
     Add javadocs to refineReport explaining parameters and return value, avoiding subjective language.
     Part 3:
     Ensure no side effects (no printing to console).
     Part 4:
     Handle edge cases:
       - Null or empty draft: return "No report available"
       - Unrecognized feedback: return draft unchanged
