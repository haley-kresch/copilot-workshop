## FewShot.java

Example: Map country codes to country names

Vague Prompt:
    Add a method to map country codes to country names.

Few-Shot Prompt:
    Add a method named `getCountryName` that maps:
      'US' → 'United States'
      'FR' → 'France'
      'JP' → 'Japan'
    Then extend the mapping for other country codes.
